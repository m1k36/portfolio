# Portfolio

This repository contains the source code of my personal portfolio website.

## Tech

- Next.js 16, TypeScript, Tailwind CSS v4
- i18n: English / French
- Icons: react-icons

## Run

```bash
npm install

# Run in dev
npm run dev   # http://localhost:3000

# Run in production
npm run build
npm run start
```
