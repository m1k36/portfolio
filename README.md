# Portfolio

This repository contains the source code of my personal portfolio website.

## Tech

- Next.js 16, TypeScript, Tailwind CSS v4
- i18n: English / French
- Icons: react-icons

## Run

```bash
npm install
npm run dev   # http://localhost:8080
```
